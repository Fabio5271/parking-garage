### Project Details:
- The project uses the UTC timezone, as per the timestamp format present in the project specification (ex: "2025-01-01T12:00:00.000Z").
- The event DTO had to be adapted to match the garage simulator, which uses the timestamp format: "yyyy-MM-dd'T'HH:mm:ss", differing slightly from the specification.
- Aside from this, the project was engineered to follow all business rules, standards, and evaluation criteria in the specification.
